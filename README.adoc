= Patch Bridge
Jonathan D.A. Jewell <j.d.a.jewell@open.ac.uk>
:toc: macro

**Draft CVE-mitigation design and extraction target from `panic-attacker`.**

Patch Bridge aims to close the detection-to-remediation gap in CVE lifecycle
management through reachability-aware triage, mitigation planning, and
adoption gating.

Current status:

* design/research draft, not a finished standalone tool
* no runnable Rust crate target yet
* no end-to-end Idris2 proof artefacts in this repository
* not production-ready
* not publication-ready in its current state

== Origin

Extracted from https://github.com/hyperpolymath/panic-attacker[panic-attacker] `src/bridge/` into a standalone project.
